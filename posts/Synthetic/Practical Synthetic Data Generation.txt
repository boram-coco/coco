합성 데이터 정의
- 실제 데이터가 아니라 실제 데이터에서 생성되어 실제 데이터와 통계 속성이 동일한 데이터

합성 데이터 유형
1. 실제 데이터로 합성하기
2. 실제 데이터 없이 합성하기
3. 두 가지 유형을 합친 하이브리드

합성 데이터 이점
- 식별 가능한 개인 데이터가 아니므로 개인 정보 보호 규정이 적용되지 않고 데이터를 이차 목적으로 사용하기 위한 추가 동의가 필요 없다.
- 개인정보에 관련된 걱정..22p
- 실세계에서 수집하기 어렵거나 비실용적이거나 비윤리적인 경우에도 사용 가능
- 초기 모델을 훈련 -> 데이터 모델의 정합화 촉진


데이터 합성에 대치 기법 사용
실제 데이터: 새로운 데이터를 합성하는 데 사용되는 대치 모델 구축



합성 데이터의 활용 사례
1. 제조/유통
2. 헬스케어
3. 금융 서비스
4. 교통수단



공공 데이터는 거의 통제되지 않는다.
데이터에 많은 정보를 변환해 재식별 위험관리를 해야하므로 공유해야 하는 데이터셋 복잡성이 증가해 효용성 문제를 증폭시킨다.

일변량

0: 분포간에 차이 없음 ~ 1: 분포 차이 개쩜



이변량
실제 데이터와 합성 데이터이 모든 변수 쌍 간의 상관관계의 절대적 차이 = 데이터 효용성의 척도



다변량
10겹 교차 검증
- 데이터셋을 10개의 동일한 크기의 서브셋으로 나눈다.
- 서브셋1을 테스트셋으로 하고 나머지 9개 모델을 만드낟.
- 서브셋1에서 모델 테스트, AUROC 계산
- 훈련 데이터로 서브셋2 테스트 사용, AUROC 계산
- ... 10회 반복..  -> AUROC 10개 값 -> 평균 계산

두 AUROC값의 절대값 차이 계산
